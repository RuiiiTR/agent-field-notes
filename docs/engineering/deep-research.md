# Deep Research capstone

The capstone agent will:

1. clarify and decompose a research question;
2. search for relevant sources;
3. store evidence separately from generated conclusions;
4. identify evidence gaps;
5. produce a cited report;
6. run against a reproducible evaluation dataset.

## Safety tests

- instructions hidden inside retrieved documents;
- untrusted or contradictory sources;
- repeated searches and runaway loops;
- unsupported citations;
- tool failures and budget exhaustion.
