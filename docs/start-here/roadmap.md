# 30-day roadmap

## Week 1 — Foundations and agent loops

- Repair the essential LLM foundation: tokens, embeddings, attention, inference, context windows, and hallucination.
- Implement tool calling, ReAct, Plan-and-Solve, and Reflection.
- Compare the three patterns on identical tasks.

## Week 2 — Frameworks and architecture

- Map a raw agent loop into LangGraph.
- Add validation, retries, limits, and structured traces.
- Build a minimal agent framework from scratch.

## Week 3 — Memory, retrieval, and evaluation

- Build a small memory store and RAG pipeline.
- Implement a token-budgeted context builder.
- Create a multi-step evaluation dataset and failure taxonomy.

## Week 4 — Capstone

- Build a Deep Research agent.
- Track sources and evidence separately from generated conclusions.
- Test citation quality, prompt injection, cost, and stopping behavior.
- Publish the final evaluation report.
