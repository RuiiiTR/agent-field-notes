# Learning method

Every topic follows the same feedback loop:

1. **Read** one primary source tied to the current problem.
2. **Explain** the mechanism from memory.
3. **Build** its smallest useful implementation.
4. **Predict** what it will do on a defined task.
5. **Break** it using edge cases and hostile inputs.
6. **Measure** success, steps, cost, latency, and unsupported claims.
7. **Write** what the evidence supports—and what it does not.
