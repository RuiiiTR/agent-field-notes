# About this blog

I created **Agent Field Notes** to make my learning verifiable.

Each substantial post should contain:

- an explanation in my own words;
- a working implementation;
- a prediction made before the experiment;
- at least one real failure;
- measurable results;
- a distinction between observation and interpretation;
- remaining limitations and questions.

This is a learning record, not a claim of expertise. Corrections are welcome.
